    - [X] Must achieve 100% code coverage in tests

- Web App 
    - [ ] Must list downloaded files
    - [ ] Must allow any size file upload
    - [ ] Must have an upload button
    - [ ] Must show upload progress
    - [ ] Must be able to upload by drag and drop
